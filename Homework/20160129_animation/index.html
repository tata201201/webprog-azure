วิธีการทำ Animation จอดรถ โดย HTML5 และ CSS

ตัวอย่าง

ขั้นตอนการสร้าง

  1. สร้างไฟล์ HTML
    2. <!DOCTYPE html>
    <html>
    <head>
    <meta charset="UTF-8">
    <title>CSS Animation Tutorial</title>
    </head>
    <body>
    </body>
    </html>
  2. แทรกรูปภาพลงไปใน tag body
    3. <img src="car.PNG">
  3. ตั้ง id ให้กับรูปภาพที่แทรกไว้ เช่นในที่นี้ตั้งว่า mycar
    4. ตัวอย่างรูปภาพ :
    <img src="car.PNG" id="mycar">
  4. แทรก css เพื่อให้รถสามารถเคลื่อนที่ได้
    สังเกตที่ #mycar นั่นคือ id ของรูปรถที่เราได้สร้างไว้นั้นเอง
    5. <style type="text/css">
    #mycar {
    position: relative;
    -webkit-animation-name: example;
    -webkit-animation-duration: 6s;
    animation-iteration-count: infinite;
    }

    @-webkit-keyframes example {
    0% {left:0px; top:0px;}
    100% {left:500px; top:0px;}
    }
    </style>

Source Code ทั้งหมด

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>CSS Animation Tutorial</title>
<style type="text/css">
#mycar {
position: relative;
-webkit-animation-name: example;
-webkit-animation-duration: 6s;
animation-iteration-count: infinite;
}

@-webkit-keyframes example {
0% {left:0px; top:0px;}
100% {left:calc(100% - 166px); top:0px;}
}
</style>
</head>
<body>
<img src="car.jpg" id="mycar">
</body>
</html>
